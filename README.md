# 🎮 Word Game

A web-based word game built with React. Players can enjoy an interactive word challenge through a responsive and modern user interface.

## 🌐 Live Demo

https://spellxspell.vercel.app

## 📖 About

Word Game is a web application developed as a personal project to practice frontend development with React and integrate a backend service using Supabase.

The project focuses on creating an interactive user experience with responsive layouts and efficient state management.

## ✨ Features

### Player

- User registration and login
- Adventure mode gameplay
- Character selection and management
- Purchase and upgrade characters
- Inventory and potion management
- View game results and rewards
- Dictionary library
- Monster library
- Word history log
- Most frequently used words
- Search and filter game data

### Administrator

- Manage dictionaries
- Manage stages
- Manage monsters
- Manage character data
- Configure monster spawning
- Manage user permissions
- Control server status

## 🛠 Tech Stack

### Frontend

- React
- CSS
- Material UI (MUI)

### Backend

- Supabase

### Deployment

- Vercel

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/word-game.git
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```
